# Diffcheck

A simple, private diff checker

## Installation

```bash
npm install
```

## Development

Run the development server and watch for changes:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

## Build

Create a production build:

```bash
npm run build
```
